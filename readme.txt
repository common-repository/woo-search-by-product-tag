Simple lightweight plugin that implements searching products by tags in woocommerce.

== Description ==

It is quite surprising but by default woocommerce does not search products by product tags. This is a simple plugin which adds Tag search functionality in woocommerce.
There are no settings in this plugin and this will start working automatically after you install and activate it


== Installation ==

1. Upload `woo-searchbyproducttag` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
